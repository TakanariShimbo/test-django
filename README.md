## 🐍Conda

- conda or miniconda is required

### Create venv

```sh
conda create -n django_sudoku_app python=3.10
```

### Activate venv

```sh
conda activate django_sudoku_app
```

### Install libs

```sh
# Please activate venv before executing it.
pip install -r requirements.txt
```
